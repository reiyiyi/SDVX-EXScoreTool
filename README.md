# SDVX-EXScoreTool

[SDVX-EX スコアツール](https://sdvx-exscoretool.herokuapp.com)は、本サービスの利用者が登録した EX スコアデータをもとに、武器曲やランキングを求めて表示する機能などを提供する Web アプリケーションです。
