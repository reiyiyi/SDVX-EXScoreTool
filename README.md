# SDVX-EXScoreTool
<a href="https://sdvx-exscoretool.herokuapp.com">SDVX-EXスコアツール</a>は、本サービスの利用者が登録したEXスコアデータをもとに、武器曲やランキングを求めて表示する機能などを提供する
Webアプリケーションです。
